# Система ранений (Wound System)

Система ранений реализована по принципам Space Station 14: урон накапливается по типам, суммируется и вычитается из максимального здоровья. Разные типы ран лечатся разными средствами.

---

## Основные понятия

### Типы урона (DamageType)

| Тип | Название | Источники | Особенности |
|-----|----------|-----------|-------------|
| `Slash` | Порезы | Лезвия, когти, острые предметы | Может вызвать **кровотечение** при сильном ударе |
| `Blunt` | Ушибы | Удары тупыми предметами, падения, кулаки | Не вызывает кровотечение |
| `Burn` | Ожоги | Огонь, кипяток, кислоты | Не вызывает кровотечение |
| `Exhaustion` | Истощение | Голод, жажда, общее истощение | Копится от метаболизма, снимается отдыхом и хорошим состоянием |

### Как урон влияет на здоровье

```
Здоровье = MaxHealth - (Slash + Blunt + Burn + Exhaustion)
```

Урон по каждому типу — это число, которое **накапливается**. Например, если персонаж получил 20 единиц пореза и 15 единиц ушиба, а его MaxHealth = 100, то:

```
Health = 100 - (20 + 15 + 0 + 0) = 65
```

Это работает **как в SS14**: здоровье не уменьшается напрямую, а рассчитывается из общего урона. Лечение, естественное восстановление и метаболизм меняют отдельные типы урона, а здоровье пересчитывается автоматически.

## Кровотечение (Bleeding)

Кровотечение возникает **только от порезов** и **только при достаточной силе единичного удара**.

### Когда начинается кровотечение

```
Единичный урон порезом >= BleedThreshold (по умолчанию 15)
```

**Важно:** много мелких порезов (например 10 ударов по 5 урона = 50 Slash) **не вызовут** кровотечение. Кровотечение зависит от силы **единичного** удара, а не от суммы.

### Механика кровотечения

Каждый сильный порез создаёт отдельную запись кровотечения (`BleedingEntry`) с параметрами:

- **Rate** — скорость кровопотери (HP урона Slash в секунду)
  - `Rate = сила_удара × BleedRateMultiplier (0.3)`
  - Пример: удар в 20 создаёт кровотечение с Rate = 6.0/с
- **NaturalClotRate** — скорость свёртывания (снижение Rate в секунду)
  - Равна `BaseClotRate (0.4)` для обычных кровотечений
  - Равна **0** для слишком сильных (Rate ≥ `UnstoppableBleedRate`, по умолчанию 5.0)

### Естественная остановка

Каждый кадр Rate кровотечения снижается на `NaturalClotRate × deltaTime`. Когда Rate достигает 0 — кровотечение прекращается.

**Исключение:** если начальный Rate ≥ `UnstoppableBleedRate` (5.0), кровотечение **не останавливается само**. Его можно остановить только перевязкой (бинтом).

### Урон от кровотечения

Кровотечение добавляет урон типа `Slash` каждый кадр:

```
SlashDamage += TotalBleedRate × deltaTime
```

Это значит что кровотечение постепенно увеличивает общий урон от порезов.

### Примеры

| Удар | Кровотечение? | Rate | Остановится само? |
|------|--------------|------|-------------------|
| 10 Slash | Нет (< 15) | — | — |
| 15 Slash | Да | 4.5/с | Да (~11 сек) |
| 20 Slash | Да | 6.0/с | **Нет** (Rate > 5.0) |
| 30 Slash | Да | 9.0/с | **Нет** |

## Компоненты

### WoundComponent (`"wounds"`)

Основной компонент. Хранит накопленный урон по типам и список активных кровотечений.

**Файл:** `MTEngine/Wounds/WoundComponent.cs`

**Поля прототипа:**

| Поле | Тип | По умолч. | Описание |
|------|-----|-----------|----------|
| `slashDamage` | float | 0 | Начальный урон порезами |
| `bluntDamage` | float | 0 | Начальный урон ушибами |
| `burnDamage` | float | 0 | Начальный урон ожогами |
| `exhaustionDamage` | float | 0 | Начальный урон истощением |
| `bleedThreshold` | float | 15 | Мин. сила пореза для кровотечения |
| `bleedRateMultiplier` | float | 0.3 | Множитель силы → Rate |
| `baseClotRate` | float | 0.4 | Скорость свёртывания |
| `unstoppableBleedRate` | float | 5.0 | Порог неостанавливаемого кровотечения |

**Пример прототипа:**

```json
{
  "components": {
    "wounds": {}
  }
}
```

Все поля имеют разумные значения по умолчанию — пустой объект `{}` достаточен.

### HealingComponent (`"healing"`)

Ставится на предметы-лечилки. Позволяет лечить определённый тип урона и/или останавливать кровотечение.

**Файл:** `MTEngine/Wounds/HealingComponent.cs`

**Поля прототипа:**

| Поле | Тип | По умолч. | Описание |
|------|-----|-----------|----------|
| `damageType` | DamageType | Slash | Тип урона для лечения |
| `healAmount` | float | 20 | Сколько урона снимает за раз |
| `stopBleedAmount` | float | 0 | Сколько HP/с кровотечения останавливает |
| `useLabel` | string | "Лечить" | Название действия в меню |

**Использование:** предмет должен быть в руке. ПКМ на раненого → появляется действие лечения. После использования предмет расходуется (1 стак).

## Лечебные предметы

### Тряпичная повязка (bandage)

```json
{
  "healing": {
    "damageType": "Slash",
    "healAmount": 25,
    "stopBleedAmount": 3.0,
    "useLabel": "Перевязать"
  }
}
```

Лечит **порезы** на 25 единиц. Останавливает кровотечение на 3.0/с. Основное средство от порезов и кровопотери.

### Травяная мазь (herbal_salve)

```json
{
  "healing": {
    "damageType": "Burn",
    "healAmount": 30,
    "stopBleedAmount": 0,
    "useLabel": "Наложить мазь"
  }
}
```

Лечит **ожоги** на 30 единиц. Мазь из целебных трав и мёда — средневековый рецепт (исторически использовались мёд, алоэ, окопник).

### Припарка (poultice)

```json
{
  "healing": {
    "damageType": "Blunt",
    "healAmount": 25,
    "stopBleedAmount": 0,
    "useLabel": "Наложить припарку"
  }
}
```

Лечит **ушибы** на 25 единиц. Влажная припарка из глины и трав (исторически: глиняные компрессы с травами для синяков и отёков).

## API для нанесения и лечения урона

### Нанесение урона

```csharp
// Нанести 25 единиц режущего урона
WoundComponent.ApplyDamage(entity, DamageType.Slash, 25f);

// Нанести 10 единиц ожога
WoundComponent.ApplyDamage(entity, DamageType.Burn, 10f);

// Нанести 15 единиц ушиба
WoundComponent.ApplyDamage(entity, DamageType.Blunt, 15f);
```

Возвращает `true` если урон был нанесён (у цели есть WoundComponent).

Если порез >= `BleedThreshold`, автоматически создаётся кровотечение.

### Лечение урона

```csharp
// Вылечить 20 единиц пореза
float healed = WoundComponent.HealDamage(entity, DamageType.Slash, 20f);

// Остановить кровотечение на 3.0/с
float stopped = WoundComponent.StopBleeding(entity, 3f);
```

`HealDamage` возвращает реально вылеченное количество (может быть меньше, если урона меньше чем healAmount).

`StopBleeding` убирает кровотечения начиная с самых слабых, возвращает суммарное снижение Rate.

### Чтение состояния

```csharp
var wounds = entity.GetComponent<WoundComponent>();

float totalDamage = wounds.TotalDamage;       // Суммарный урон
float slashDmg = wounds.SlashDamage;           // Урон порезами
float exhaustionDmg = wounds.ExhaustionDamage; // Урон истощением
float bleedRate = wounds.TotalBleedRate;       // Суммарная скорость кровотечения
bool bleeding = wounds.IsBleeding;              // Есть ли кровотечение
```

## Метаболизм и регенерация

Голод и жажда больше не бьют `Health` напрямую. Вместо этого при критических состояниях метаболизм добавляет урон типа `Exhaustion`:

```csharp
WoundComponent.ApplyDamage(entity, DamageType.Exhaustion, starvationPlusDehydration);
```

Если у персонажа всё хорошо:

- сытость >= `naturalRegenThreshold`
- жажда >= `naturalRegenThreshold`
- нет активного кровотечения
- нет критической нужды в туалет

то включается естественная регенерация. Она сначала снимает `Exhaustion`, а затем понемногу заживляет обычные раны.

**Поля `MetabolismComponent`:**

| Поле | Тип | По умолч. | Описание |
|------|-----|-----------|----------|
| `naturalRegenRate` | float | 1.2 | Скорость естественной регенерации в сек |
| `naturalRegenThreshold` | float | 70 | Порог сытости/жажды для старта регена |
| `exhaustionRecoveryRate` | float | 3.5 | Скорость снятия истощения после нормализации состояния |
| `exhaustionRecoveryThreshold` | float | 45 | Минимум сытости/жажды для начала снятия истощения |

Истощение теперь уходит заметно легче:

- если голод или жажда снова критические, `Exhaustion` продолжает копиться;
- если еда и питьё уже **переварились** и `Hunger`/`Thirst` поднялись хотя бы до `exhaustionRecoveryThreshold`, `Exhaustion` начинает быстро спадать;
- если состояние уже хорошее (`naturalRegenThreshold`), поверх этого включается и обычная регенерация здоровья.

## WoundSystem

**Файл:** `MTEngine/Wounds/WoundSystem.cs`

Система обновляется каждый кадр и делает три вещи:

1. **Тик кровотечений** — каждое активное кровотечение добавляет `Rate × dt` к `SlashDamage`
2. **Свёртывание** — Rate каждого кровотечения уменьшается на `NaturalClotRate × dt`. Если Rate ≤ 0, кровотечение удаляется
3. **Синхронизация HP** — `Health = MaxHealth - TotalDamage`

WoundSystem зарегистрирована **перед** HealthSystem, чтобы HealthSystem могла корректно обработать смерть.

## Взаимодействие: Осмотр ранений

WoundComponent реализует `IInteractionSource` и добавляет действие **"Осмотреть ранения"** в контекстное меню (ПКМ).

При нажатии открывается XML-окно с информацией:

- Список типов урона с числовыми значениями (цвет зависит от тяжести)
- Общий суммарный урон
- Информация о кровотечении (если есть) с пометкой `[СИЛЬНОЕ!]` для неостанавливаемых
- Текущее здоровье

**Цветовая индикация:**

| Урон | Цвет |
|------|------|
| < 20 | Жёлтый (лёгкие) |
| 20–50 | Оранжевый (средние) |
| > 50 | Красный (тяжёлые) |

## Как добавить новый тип урона

1. Добавить значение в `enum DamageType` в `WoundComponent.cs`
2. Добавить поле хранения (напр. `PoisonDamage`) с атрибутами `[SaveField]` и `[DataField]`
3. Обновить `GetDamage()`, `SetDamage()`, `TotalDamage`
4. Обновить UI осмотра в `BuildExamineXml()`
5. При необходимости — создать новый лечебный предмет с `"damageType": "Poison"`

## Как создать новый лечебный предмет

Создать папку в `SandboxGame/Content/Prototypes/Medical/` со `sprite.png` и `proto.json`:

```json
{
  "id": "my_medicine",
  "name": "My Medicine",
  "category": "entity",
  "components": {
    "sprite": { "source": "sprite.png", "width": 32, "height": 32, "layerDepth": 0.5 },
    "transform": { "x": 0, "y": 0 },
    "item": {
      "name": "Моё лекарство",
      "size": "Small",
      "slots": 1,
      "tags": ["medical"]
    },
    "info": {
      "description": "Описание лекарства."
    },
    "healing": {
      "damageType": "Burn",
      "healAmount": 50,
      "stopBleedAmount": 0,
      "useLabel": "Применить"
    }
  }
}
```
