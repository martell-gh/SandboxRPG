# Localization

Локализации лежат в `SandboxGame/Content/Localization`.

Каждый язык - отдельная папка:

```text
Localization/
  RU/
    meta.json
    base.lcl
  EN/
    meta.json
    items.lcl
```

`meta.json` нужен для списка языков в настройках:

```json
{
  "name": "Русский"
}
```

Файлы `.lcl` можно называть как угодно. Загружаются все строки вида:

```text
--replic-man-hello: "Привет"
--item-name-apple: "Яблоко"
```

Ключ всегда начинается с `--`. В прототипах, UI и редакторских полях можно писать сам ключ вместо текста, например:

```json
{
  "item": {
    "name": "--item-name-apple"
  }
}
```

Порядок поиска строки:

1. текущий язык;
2. `EN`;
3. `RU`;
4. пустая строка, если ключ нигде не найден.

Новый язык добавляется созданием новой папки с `meta.json`; после запуска он появится в настройках в списке языков.
